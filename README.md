# Gas Station Revenue & Risk Decision System

End-to-end operational decision system for a multi-service fuel station.

## Layers

1. Simulation – Operational fuel, services, debt
2. Business Logic – Revenue & expected cash reconciliation
3. Preprocessing – Feature engineering
4. Anomaly Detection – Isolation Forest-based risk flagging
5. Evaluation – Business impact & model metrics

This project transforms paper-based operational records into a structured, decision-ready analytics system.
